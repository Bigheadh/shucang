# AI 变更记录模板

> **核心理念**：AI 每次修改代码后，必须输出这份变更记录。
> **原则**：宁可过细，不可遗漏。每一个文件的每一处改动都要记录。

---

## 提交信息

| 字段 | 内容 |
|------|------|
| 日期 | YYYY-MM-DD HH:mm |
| 任务 | 一句话说明本次要做什么 |
| AI 模型 | 例如 claude-sonnet-4, deepseek-v4 等 |
| 涉及文件数 | N 个 |
